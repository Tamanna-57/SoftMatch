# SoftMatch — Quick Start

## 1. Install & run backend
```bash
cd backend
npm install
npm start
```

## 2. Install & run frontend (new terminal)
```bash
cd frontend
npm install
npm run dev
```

## 3. Open http://localhost:5173

The app works fully in demo mode without the backend.
Start the backend to enable real multi-user matching.

See frontend/README.md for full documentation.
